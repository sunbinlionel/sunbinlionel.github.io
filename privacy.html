3. Account Deletion / 账号删除

🔴 Apple App Store Requirement / Apple应用商店要求

As required by Apple App Store guidelines, users can delete their account at any time.

根据Apple应用商店指南要求，用户可以随时删除其账号。

3.1 How to Delete Your Account / 如何删除账号

You can delete your account through the following steps:

您可以通过以下步骤删除账号：

  1. Open the app and log in / 打开应用并登录
  2. Go to User Center (bottom right tab) / 进入用户中心（右下角标签）
  3. Tap "Account & Security" / 点击"账号与安全"
  4. Tap "Delete Account" / 点击"删除账号"
  5. Read the warning information carefully / 仔细阅读警告信息
  6. Confirm deletion / 确认删除

3.2 What Happens When You Delete Your Account / 删除账号后会发生什么

Upon account deletion, the following actions will be taken immediately:

删除账号后，将立即执行以下操作：

  • Personal Information Removed / 个人信息将被删除
    • Username, email, and profile data will be permanently deleted / 用户名、邮箱和个人资料将被永久删除
    • Your account will be deactivated and cannot be used to log in / 账号将被停用，无法再登录
  • Analysis History Anonymized / 分析历史将被匿名化
    • Your assessment results will be anonymized (personal identifiers removed) / 您的测评结果将被匿名化（移除个人标识）
    • Anonymized data may be retained for statistical analysis and service improvement / 匿名数据可能被保留用于统计分析和服务改进
  • Purchase Credits Forfeited / 购买次数将被放弃
    • Any remaining analysis credits will be lost / 剩余的分析次数将丢失
    • No refunds will be issued for unused credits / 未使用的次数不予退款
  • Account Cannot Be Recovered / 账号无法恢复
    • This action is permanent and irreversible / 此操作是永久性的，不可撤销
    • You will need to create a new account to use the app again / 如需再次使用应用，需要注册新账号

3.3 Data Retention After Deletion / 删除后的数据保留

After account deletion:

账号删除后：

  • Immediately Deleted / 立即删除: Personal identifiable information (username, email, password) / 个人身份信息
  • Retained Up to 90 Days / 最多保留90天: Anonymized analytics data for service improvement / 用于服务改进的匿名分析数据
  • Retained Up to 7 Years / 最多保留7年: Transaction records as required by law / 法律要求保留的交易记录

3.4 Important Notes / 重要说明

  • Irreversible: Account deletion is permanent and cannot be undone / 不可逆：账号删除是永久性的，无法撤销
  • No Refunds: Deletion does not entitle you to a refund for any purchases / 不予退款：删除账号不会退还任何购买费用
  • Immediate Processing: Account deletion is processed immediately / 立即生效：账号删除立即处理
  • New Account Required: You must create a new account to use services again / 需要新账号：需注册新账号才能再次使用服务

4. Data Security / 数据安全

We implement industry-standard security measures to protect your data:

我们实施行业标准的安全措施来保护您的数据：

  • Encryption: All data transmitted between the app and our servers is encrypted using HTTPS/TLS / 加密传输：所有数据通过HTTPS/TLS加密传输
  • Password Security: Passwords are hashed and salted using bcrypt (12 rounds) / 密码安全：密码使用bcrypt（12轮）哈希和加盐
  • Server-Side Validation: Purchase receipts are verified on our servers / 服务器端验证：购买收据在服务器端验证
  • Access Control: Strict authentication and authorization controls / 访问控制：严格的身份验证和授权控制
  • Regular Security Audits: We conduct regular security reviews / 定期安全审计：我们进行定期安全检查
